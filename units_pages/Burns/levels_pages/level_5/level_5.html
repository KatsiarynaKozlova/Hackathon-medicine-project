Memory Game

Перед вами необычная версия Memory Game. Необходимо выбрать попарно карточки, которые образуют верное утверждение по теме Ожоги (Пр.: "Правило Ладони" - "Площадь ладони = 1% от тела").

Для успешного прохождения уровня переверните все карточки, чтобы не осталось фиолетовых рубашек.
